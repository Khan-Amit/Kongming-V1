# 🔐 Kongming Wallet

## Unhackable · Device‑Bound · Admin‑Approved · CSFS Storage

Kongming Wallet is a secure, private cryptocurrency wallet that requires:
- **Phone authentication** (SMS OTP)
- **Administrator approval** (only approved users can access)
- **Device binding** (wallet tied to your specific phone)
- **CSFS storage** (cyclic append‑only ledger)

## Features

- 📱 Register with phone number
- 🔐 OTP verification (simulated, can be upgraded to real SMS)
- 👑 Admin approval panel (only you can approve new users)
- 💰 Send / receive CSFS coins
- 📦 CSFS circular storage (desire‑based pruning)
- 🔒 Device‑bound encryption

## How to Use

### For Users
1. Open the app → enter phone number
2. Enter OTP (simulated)
3. Wait for admin approval
4. Once approved, wallet unlocks
5. Send / receive CSFS coins

### For Admin
1. Open `admin.html` (only you have access)
2. See pending requests
3. Approve or deny users
4. Approved users can immediately use the wallet

## Installation (PWA)

1. Open `https://khan-amit.github.io/Kongming-Wallet/` on your phone
2. Tap three dots → Add to Home screen → Install
3. The app works offline
